# create-reducer-app

创建 Redux 通用脚手架

## 使用的模块

```
ejs `模板引擎`
inquirer `用户输入配置`
fs-extra(或者shelljs) `文件移动`
```

## node 版本

```
v16.4.0(这里使用nvm管理node版本)
```

### git 标准提交

```
[cz-cli](https://github.com/commitizen/cz-cli)
[git-cz](https://github.com/streamich/git-cz)
```

#### 使用create-reducer-app

```bash
cnpm i create-reducer-app -g
```
